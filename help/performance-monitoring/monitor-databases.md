# 监测数据库

您的 Campaign 实例的所有资源、工作流程和数据都存储在数据库中。数据库达到其最大容量可能会导致实例出现问题（无法登录、无法发送电子邮件等）。 因此，必须定期监测实例的数据库以确保最佳性能。

了解如何使用 [!DNL Control Panel].
